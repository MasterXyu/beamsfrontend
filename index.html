It's beautiful. It's Beams.

Beams is an open front-end Web framework created primarily with video-heavy sites in mind. It should also integrate well with other types of Web projects that require beautiful display across different types of devices.

Key component of the project so far is a responsive layout written in CSS3 using a gutterless 40-column 4,000-pixel grid made for livingroom-sized screens. It also scales down elegantly to the size of a smartphone screen or anything inbetween.

That responsiveness is built on simple round-number breakpoints rather than media queries written for particular manufacturers' screen sizes. We believe this will make the framework more future-proof in case of hardware changes, which seem to happen rapidly these days.

Every column here is 100 pixels wide. That's it. There are no gutters, but there is a 10px padding internally within each column.

We've also included a copy of the compressed jQuery v2.1.0 file in this project so Beams implementers can quickly access that popular library from their own servers. Other JavaScript enhancements later so that Beam becomes an even more of a full-fledged development tool.

Will certainly be adding integration with YouTube, Vimeo and Kaltura APIs for video players, etc, in short order. Of course, we're going to need that for the sake of video integration.
