Go ahead. Build it with Beams.

Beams is a front-end Web framework designed primarily for multimedia-heavy sites. It is the centerpiece of a suite of open-source modules for developers who need to support video, photos, and other content across the widest range of screens and use cases possible.

The Beams suite is currently v0.1.0, an alpha release. The modules are still a bit rough around the edges, to be sure. But we believe we at least have a good start here, something we can show to other developers, begin to build a community around, and start iterating quickly.

A quick summary of the modules, including a link to each one's code repo on GitHub:

  • Beams. The flagship of the suite, this is a front-end Web framework built in HTML5, CSS3, and jQuery. The visual design is based on a responsive 20-column, 4,000-pixel grid -- approximately the physical screen size of a 48-inch TV. Yet the framework also scales down smoothly to any other size from desktop to tablet to smartphone as well.
  • Beams Server. A simple server written in Node.js/Express for running feature-rich sites created with the front-end module.
  • Beams Ads. An optional jQuery plugin for showing interstitial display ads. You can also control how often a user sees an ad and track views of the ads with Google Analytics, all while maintaining responsive display across different screen sizes for your sponsors too.

We're glad to get pull requests, bug reports, or any other feedback you have about Beams. Drop us an email anytime or visit us on GitHub.
